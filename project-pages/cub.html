Cub: Self-Care Pet
Project Details / Background
Cub: Self-Care Pet is an innovative iOS application designed to encourage and gamify personal well-being and self-care routines. As a key developer on this project, I focused on creating an engaging and interactive user interface that motivates users to complete daily tasks and build healthy habits.
The app features a virtual pet that users care for by completing real-life self-care tasks. I implemented core functionalities such as habit tracking, reward systems, and personalized goal setting. The challenge was to create a seamless blend of gamification elements with practical self-care tools, ensuring the app remained both fun and beneficial for users' well-being.
