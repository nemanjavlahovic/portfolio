Meridian Sport Srbija
Project Details / Background
Meridian Sport Srbija is a comprehensive sports news and live scores application tailored for the Serbian market. As a key iOS developer on this project, I focused on creating a fast, responsive, and feature-rich app that caters to sports enthusiasts across the country.
The app provides real-time updates on various sports, including football, basketball, tennis, and more. I implemented core functionalities such as live score tracking, push notifications for game updates, and personalized news feeds based on user preferences. A major challenge was ensuring smooth performance while handling large amounts of real-time data and providing a seamless user experience across different iOS devices.
