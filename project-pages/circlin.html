Circlin: Buy & Sell Fashion
Project Details / Background
Role: iOS Architect & Backend Engineer (Contract)
Circlin is a video-first peer-to-peer marketplace for pre-loved fashion launching in Switzerland. The platform differentiates itself with video listings that build trust and an AI selling assistant that automates tedious tasks like writing descriptions and detecting products.
I was responsible for architecting the iOS application from the ground up and implementing the backend services using GraphQL. This involved designing a scalable mobile architecture that supports video-first content delivery, integrating AI-powered features for automated listing creation, and building a robust GraphQL API layer to handle real-time marketplace operations including secure payment processing through Stripe and shipping logistics via Swiss Post.
